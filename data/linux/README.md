# "腺"而易见 - 训练专用版本

睑板腺 MGD 智能辅助诊疗系统 (V100 32GB 优化训练版)

---

## ⚡ 优化特点 (针对 NVIDIA V100 32GB)

- 🚀 **默认批次大小**: 8 (充分利用显存)
- ⚡ **默认精度**: FP16 混合精度训练
- 📈 **训练轮数**: 100轮
- 🎯 **损失函数**: Dice+BCE 组合损失
- 🔄 **学习率调度**: CosineAnnealing 余弦退火
- 👥 **数据加载**: 8线程并行加载
- 💾 **Checkpoint**: 每10轮保存，保留最新5个

---

## 📋 概述

本系统为"腺"而易见训练引擎添加了完整的交互式控制台界面，支持预设管理、参数配置、批量训练等功能。

---

## 🚀 快速开始

### 一键启动（推荐）

在 Linux 系统上执行：

```bash
cd 腺而易见WebUI
chmod +x run.sh  # 首次使用添加执行权限
./run.sh
```

**就是这么简单！** 脚本会自动：
1. 🔄 关闭 conda 环境（执行3次）
2. 📦 检查 Python
3. 🐍 自动创建/激活 venv 虚拟环境
4. 📥 自动安装依赖
5. 🎮 让你选择启动模式

### 直接用命令行参数启动

如果不想用交互式界面，可直接传参数：

```bash
./run.sh --epochs 50 --batch-size 16
```

会自动以命令行模式运行原 train_cli.py

## 🎯 功能介绍

### 1. 主菜单

程序启动后会显示主菜单，包含以下选项：

```
请选择操作:
  1. 选用已有预设
  2. 新建自定义预设
  3. 批量训练
  4. 退出程序
```

### 2. 选用已有预设

**功能：** 从已保存的预设中选择一个进行训练

**操作流程：**
1. 程序列出所有可用预设
2. 输入预设序号选择
3. 查看预设参数
4. 自定义训练轮数
5. 可选：修改其他参数
6. 确认后开始训练

**特点：**
- 预设参数完整显示
- 支持单独修改训练轮数
- 支持修改全部参数
- 训练过程实时显示

### 3. 新建自定义预设

**功能：** 逐项配置训练参数，创建新预设

**操作流程：**
1. 逐项配置 13 个训练参数
2. 每步可输入 `0` 返回上一项修改
3. 完成后显示完整配置确认
4. 输入预设名称保存
5. 选择是否立即开始训练

**可配置参数：**

| 参数 | 类型 | 说明 |
|------|------|------|
| 训练轮数 | 数值 | 1-1000 |
| 批次大小 | 数值 | 1-32 |
| 学习率 | 数值 | 0.000001-0.1 |
| 优化器 | 选择 | Adam/AdamW/SGD |
| 学习率调度器 | 选择 | 5种可选 |
| 损失函数 | 选择 | 4种可选 |
| 训练精度 | 选择 | fp16/fp32 |
| 数据增强 | 选择 | 开启/关闭 |
| 数据加载线程数 | 数值 | 0-32 |
| 保存频率 | 数值 | 1-100 |
| 保留Checkpoint数量 | 数值 | 1-20 |
| 权重衰减 | 数值 | 0-0.1 |
| 正负样本权重 | 数值 | 0.1-10 |

**全局回退功能：**
- 在配置任何参数时，输入 `0` 可返回上一项重新配置
- 确认页面支持重新配置或返回主菜单

### 4. 批量训练

**功能：** 选择多个预设，按顺序自动执行训练

**操作流程：**
1. 显示所有预设列表
2. 输入预设序号，多个用逗号分隔（如：1,3,5）
3. 确认选择的预设
4. 自动按顺序执行训练
5. 训练完成后显示结果汇总

**特点：**
- 支持任意数量预设
- 自动依次执行
- 每个预设训练结果独立显示
- 最终汇总所有结果

## 💾 预设管理

### 默认预设

首次运行时会自动创建 3 个默认预设：

1. **V100 32GB 优化配置** - 针对 V100 显卡优化
2. **快速测试配置** - 小批次、少轮数，用于快速验证
3. **高精度训练配置** - 高精度、长时训练

### 预设存储

- 所有预设保存在 `training_presets.json`
- JSON 格式，可手动编辑（不推荐）
- 预设数量无限制
- 支持重名覆盖

将训练数据放入以下目录：

```
data/
├── images/          # 原始红外图像 (jpg/png)
```

## 💻 训练命令

### 快速训练（默认V100优化参数）

```bash
python3 train_cli.py
```

### 自定义参数示例

```bash
# 200轮训练，批次16
python3 train_cli.py --epochs 200 --batch-size 16

# 断点续训
python3 train_cli.py --resume --epochs 150

# 禁用数据增强
python3 train_cli.py --no-augmentation

# FP32精度训练
python3 train_cli.py --precision fp32

# 使用SGD优化器
python3 train_cli.py --optimizer SGD --lr 0.01
```

### 完整参数列表

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--epochs`, `-e` | 100 | 训练轮数 |
| `--batch-size`, `-b` | 8 | 批次大小 |
| `--lr` | 0.001 | 学习率 |
| `--optimizer` | AdamW | 优化器 (Adam/AdamW/SGD) |
| `--scheduler` | CosineAnnealing | 学习率调度器 |
| `--loss` | Dice+BCE | 损失函数 |
| `--precision` | fp16 | 训练精度 (fp16/fp32) |
| `--augmentation` | True | 数据增强 |
| `--no-augmentation` | - | 禁用数据增强 |
| `--resume` | - | 断点续训 |
| `--num-workers` | 8 | 数据加载线程数 |
| `--save-freq` | 10 | 保存频率 |
| `--max-keep` | 5 | 保留checkpoint数量 |

## 🔧 故障排除

### GPU显存不足

虽然已针对V100 32GB优化，但如果仍OOM：
- 减小批次大小: `--batch-size 4`
- 或启用分块训练: `--tile-mode`

### 数据集加载失败

- 检查 `data_loader.py` 中的 `data_dir` 路径
- 确保 `data/images/`, `data/gland_masks/`, `data/eyelid_masks/` 下有数据
- 确保图像和标注文件数量一致且文件名对应

### 虚拟环境问题

删除 `./venv` 目录重新运行 `./run.sh`

### 预设文件损坏

**解决：** 删除 `training_presets.json`，重启程序会重新创建默认预设

### 参数配置错误

**解决：** 在配置过程中随时输入 `0` 返回修改，或在确认页选择重新配置

### 批量训练中断

**解决：** 已完成的训练结果已保存，重新选择剩余预设继续
